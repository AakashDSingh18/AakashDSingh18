🎨 Creative Contribution Calendar

Interactive visualization of GitHub contributions

✨ Features

🎯 Interactive Calendar

Hover over any day to see detailed contribution information with smooth animations and dynamic visual feedback.

📊 Real-time Stats

Displays your GitHub statistics including public repositories, followers, and following count updated in real-time.

🎨 Custom Styling

Beautiful gradient effects, smooth transitions, and glowing effects create an engaging visual experience.

📱 Responsive Design

Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices.

⚡ Performance

Optimized animations and lazy loading ensure smooth performance across all modern browsers.

🔄 Auto-Update

Automatically fetches the latest GitHub data and updates your contribution calendar periodically.

🚀 How to Use

Getting Started

  • The calendar displays your GitHub contributions for the current year
  • Color intensity indicates the number of contributions on each day
  • Hover over any square to see the exact contribution count
  • Check the stats section to see your GitHub metrics
  • The visualization updates automatically throughout the year
